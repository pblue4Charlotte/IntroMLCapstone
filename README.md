Housing Price Prediction

This repository contains the code (.py files) and resources (train.csv) to predict housing prices using the Kaggle House Prices Dataset.

Classic Models:

model_classic_ridge.py: Ridge Regression

model_classic_lasso.py: Lasso Regression

model_classic_polynomial.py: Polynomial Regression

Paper Based Models:

model_paper_DNN.py: Deep Neural Network, V. K. Gupta, et al., (2024)
model_paper_RF.py: Random Forest, Y. Fu, (2024)

Run any script to see training results. For example:

python model_classic_lasso.py
